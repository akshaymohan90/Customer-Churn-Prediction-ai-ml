Predict whether customers will leave a telecom company.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* XGBoost

## ML Models

* Logistic Regression
* Random Forest
* XGBoost

## Evaluation Metrics

* Accuracy
* Recall
* F1 Score
* ROC-AUC

## Outcome

Built an end-to-end machine learning pipeline for customer churn prediction.
